this is a restuarant booking system using django rest framework. please refer the APIs listed below.

http://127.0.0.1:8000/auth/token/login/               -  login the user and obtain the token
http://127.0.0.1:8000/auth/token/logout/              -  logout the user and delete the token
http://127.0.0.1:8000/restaurant/api-token-auth/      -  obtain the token
http://127.0.0.1:8000/auth/users                      -  listdown the users registered in the system
http://127.0.0.1:8000/restaurant/menu/                -  list down the menu items and create the menu item
http://127.0.0.1:8000/restaurant/menu/3               -  get the partiular menu item and update it or delete it
http://127.0.0.1:8000/restaurant/booking/tables/      -  get the bookings
